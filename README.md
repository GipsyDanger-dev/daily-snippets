# Daily AI Snippets

Materi pendek harian untuk belajar AI/ML. Setiap hari ada tutorial baru yang di-auto-generate via GitHub Actions.

## Topik yang Dicover

| Kategori | Bahasa | Topik |
|----------|--------|-------|
| **Python Fundamentals** | Python | NumPy, Pandas, Matplotlib |
| **Deep Learning** | Python | PyTorch, Neural Network, CNN |
| **Machine Learning** | Python | Scikit-learn, Classification, Clustering |
| **C++ for ML** | C++ | Smart Pointers, Templates |
| **Web AI** | JavaScript | TensorFlow.js, Browser ML |
| **Computer Vision** | Python | OpenCV, Image Processing |

## Contoh Materi

Setiap tutorial mencakup:
- Penjelasan konsep
- Contoh kode lengkap
- Best practices
- Kesalahan umum
- Latihan mandiri
- Sumber belajar

## Cara Pakai

1. Buka folder `snippets/`
2. Pilih tanggal yang mau dipelajari
3. Baca dan pahami materinya
4. Kerjakan latihannya

## Auto Update

GitHub Actions generate tutorial baru setiap hari jam 19:00 WIB.

## Structure

```
daily-snippets/
├── .github/workflows/    # Auto-update automation
├── generate.py           # Tutorial generator script
├── snippets/             # Daily tutorial files
│   ├── 2026-07-17.md
│   ├── 2026-07-18.md
│   └── ...
└── README.md
```
